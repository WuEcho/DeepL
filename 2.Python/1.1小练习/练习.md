## 小练习

### 利用Python求和； 以下为通过用户输入两个数字，并计算两个数字之和


```
a = int(input("请输入第一个数字:"))
b = int(input("请输入第二个数字:"))
print("这两个数字之和为%d" %(a+b))
```

### 利用Python生成随机数

```
import random
import time
current_randow_seed = int(time.time())
random.seed(current_randow_seed)
print(random.randint(0,100))
```
